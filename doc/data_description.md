# cleaned_titanic 資料欄位說明

| 欄位名稱 | 說明 |
|----------|------|
| Survived | 是否存活（0 = 否，1 = 是） |
| Pclass | 艙等（1 = 頭等，2 = 商務，3 = 經濟） |
| Sex | 性別（male/female） |
| Age | 年齡（以中位數或平均數填補遺漏值） |
| SibSp | 同行兄弟姊妹或配偶人數 |
| Parch | 同行父母或子女人數 |
| Fare | 票價（英鎊） |
| Embarked | 登船港口（C = Cherbourg, Q = Queenstown, S = Southampton） |
